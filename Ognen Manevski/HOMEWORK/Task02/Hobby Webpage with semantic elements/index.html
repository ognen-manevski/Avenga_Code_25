My Favourite Hobby - Coding

I discovered coding because I wanted to build things on the web. Coding is a creative hobby that lets me solve problems, build projects I care about, and keep learning every day. I enjoy the mix of logic and creativity - writing code feels like solving a puzzle that results in something useful.

Small, consistent practice is what helped me improve fastest.

Why I Love Coding

Coding lets me turn ideas into working websites and tools. I like how a small change in code can immediately alter the behavior or appearance of a page - that feedback loop makes learning addictive. As a hobby, coding gives me structure (algorithms, logic) and freedom (design, UX) at the same time.

Common benefits I've found from coding as a hobby:

  • Improves problem-solving and logical thinking
  • Enables personal projects like portfolios, blogs, and small web apps
  • Provides a large, collaborative community and plenty of learning resources

If you're starting out, focus on small, practical projects - a simple webpage, a to-do app, or a personal portfolio. Those concrete goals accelerate learning more than abstract exercises.

"Building small projects taught me more than tutorials ever did."

Learning Path & Resources

When I started, I followed a simple path: learn HTML & CSS for structure and styling, add JavaScript for interactivity, and practise by building small projects. Useful resources I recommend:

  • MDN Web Docs - reference and guides
  • freeCodeCamp - hands-on exercises
  • GitHub - host projects and learn from others

Tips for steady progress: set short weekly goals, read other people's code, and keep your projects small and useful. Over time, complexity will grow naturally and you'll have a portfolio to show for your hobby.
