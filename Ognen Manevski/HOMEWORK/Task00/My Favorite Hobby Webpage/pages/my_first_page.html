My First Project: Simple Portfolio

I built a small portfolio website to practice HTML and CSS and to showcase projects. The site contains a home page, a projects list and a contact section. It was created to learn layout, navigation and basic responsive techniques.

Open my first project to see the live site.

About This Project

This project focuses on a clean layout, simple responsive menu and a project gallery.
